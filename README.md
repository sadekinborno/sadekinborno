## Currently Building

### 🔬 PERF — NLP Benchmark Audit Framework *(Final Year Design Project)*
An audit and re-annotation framework for 10 NLP classification benchmarks, investigating whether widely-used "gold standard" datasets actually deserve that label. Built with Python, and supervised at UIU's Department of CSE.

- Systematic quality analysis of annotation consistency across benchmark datasets
- Developed PERF (Performance Evaluation and Re-annotation Framework) to surface labeling errors
- Research question: *Are NLP Gold Standards Truly Golden?*

### 🤖 Arc — Personal AI Ecosystem *(Side Project)*
An Iron Man-style AI assistant with voice interaction, PC control, and task automation. Think JARVIS, built from scratch.

- FastAPI backend · ElevenLabs voice integration · Redis + Qdrant for memory
- Cloudflare Tunnels / Tailscale for secure remote access · JWT + 2FA auth
- Roadmap includes full natural language PC control and multi-agent task delegation
